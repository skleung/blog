Open Source Notes

13 January 2015 on tech

I've decided to open source my notes on this Ghost blog this quarter. Two reasons why I've come to this decision rather than the Evernote setup that I used to use in past quarters:
1. Markdown support: Ever since discovering Markdown, integrating code into notes has been a breeze.
2. Easy Deployment: One-line deployment makes it dead simple to export these notes online after publishing it to my local application:
buster generate && buster deploy

One downside is that Ghost doesn't to have centralize markdown files from posts into a single location. This makes it harder to convert the markdown files into an exportable format.

For Winter 2015, I'll be posting regularly for the following classes:

  • CS108
  • CS193p
  • CS181w
