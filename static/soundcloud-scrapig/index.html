Soundcloud Scraping

19 January 2015

After learning about cURL requests from a javascript course I took last year, I wanted to automate some track scraping with a bash script so I could listen to some SoundCloud music on my flight.

The script is up here, but unfortunately my 20 minute complimentary Wi-Fi at Chicago O'hare was up before I could use it...hopefully it helps someone else out!

There's definitely a better SoundCloud scraper out there, but this one works out of the box with a personal API_CLIENT_KEY until I run out of downloads with my personal key.
